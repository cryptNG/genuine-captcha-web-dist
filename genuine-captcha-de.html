Wie es funktioniert

Flexible Hosting-Optionen

Sie können die API entweder selbst auf Ihren eigenen Servern hosten, um vollständige Kontrolle zu haben, oder von uns hosten lassen für eine problemlose Erfahrung. Beide Optionen halten die gleichen Datenschutzstandards ein und sammeln keine Daten.

Einfache API-Anfrage

Ihre Website oder Frontend stellt eine einfache API-Anfrage an den CAPTCHA-Dienst, um ein Challenge-Bild anzufordern. Dies erfordert minimalen Code und hat keine Leistungsauswirkungen auf Ihre Anwendung.

CAPTCHA anzeigen

Das Bild wird als Base64 empfangen und kann auf jeder Seite zusammen mit einem geheimen Token angezeigt werden. Dieser Ansatz funktioniert mit jedem Frontend-Framework oder reinem HTML/CSS/JS.

Benutzer löst das CAPTCHA

Der Benutzer sieht die CAPTCHA-Herausforderung auf Ihrer Website und gibt seine Lösung ein. Dies schafft eine natürliche, intuitive Verifizierungserfahrung ohne versteckte Verfolgung oder Datensammlung.

Zur Überprüfung einreichen

Wenn der Benutzer seine Lösung einreicht, sendet Ihr Frontend sowohl die Antwort des Benutzers als auch das geheime Token an den Verifizierungsendpunkt. Dies gewährleistet die Integrität des Verifizierungsprozesses.

Verifizierungsergebnis

Der Endpunkt gibt eine einfache Erfolgs- oder Fehlerantwort zurück. Basierend auf diesem Ergebnis können Sie dem Benutzer erlauben fortzufahren oder ihn bitten, es erneut zu versuchen. Der gesamte Prozess wird mit minimalem Aufwand abgeschlossen.

Warum Genuine Captcha©

Ein revolutionärer Ansatz für CAPTCHA, der Datenschutz und DSGVO-Konformität priorisiert

Warum eine weitere Captcha-Lösung?

Typische CAPTCHA-Lösungen im Web können, obwohl sie möglicherweise effektiv bei der Unterscheidung zwischen menschlichen Benutzern und Bots sind, mehrere Datenschutzbedenken aufwerfen:

  • Die meisten sammeln Benutzerdaten wie IP-Adressen, Mausbewegungen und Surfverhalten
  • Potenziell werden Datenschutzdaten für Profilerstellung oder Werbung aggregiert
  • Verwendung von Cookies oder Tracking-Technologien und dadurch DSGVO-bezogene Bedenken
  • Viele verlassen sich auf Drittanbieterdienste, was das Risiko unbeabsichtigter Datenweitergabe birgt
  • Einige verwenden invasive Methoden wie Tastaturanschlagsüberwachung und könnten an Transparenz bezüglich Datennutzung und -speicherung mangeln
  • Die von CAPTCHA-Systemen gesammelten Daten könnten bei Missbrauch die Privatsphäre der Benutzer gefährden

Diese Probleme unterstreichen die Notwendigkeit datenschutzbewussterer CAPTCHA-Alternativen in der Websicherheit.

Warum Genuine Captcha© wählen? Ein Wendepunkt in der Benutzerverifizierung

Genuine Captcha© erweist sich als revolutionäre Lösung im Bereich der CAPTCHA-Integration, insbesondere für diejenigen, die DSGVO-Konformität und Benutzerdatenschutz priorisieren. Diese Plattform zeichnet sich durch ihre außergewöhnlichen Funktionen aus:

  • DSGVO-Konformität: Genuine Captcha© ist darauf ausgelegt, die Standards der DSGVO zu erfüllen. Durch die Wahl dieser Plattform können Integratoren diese strengen Datenschutzgesetze einhalten und rechtliche Konformität sowie Seelenfrieden gewährleisten.
  • Open-Source: Der Quellcode von Genuine Captcha© ist transparent und flexibel. Dieser Aspekt ermöglicht kontinuierliche Verbesserungen, angetrieben von einer Open-Source-Community.
  • Datenschutzzentrierte Verifizierung: Das Design von Genuine Captcha© ist das Engagement für Benutzerdatenschutz. Es verifiziert menschliche Interaktionen, ohne private Daten zu sammeln, und setzt einen neuen Maßstab in der Respektierung der Privatsphäre der Benutzer.
  • Verbesserte Benutzererfahrung: Durch den Verzicht auf die Sammlung persönlicher Daten bietet Genuine Captcha© eine respektvollere Benutzererfahrung. Dieser Ansatz adressiert und löst die Datenschutzbedenken, die üblicherweise mit traditionellen CAPTCHA-Systemen verbunden sind.
  • Innovative Verifizierung: Genuine Captcha© zeichnet sich durch die Verifizierung menschlicher Interaktionen mit einem innovativen Ansatz aus. Dies gewährleistet einen nahtlosen und benutzerfreundlichen Verifizierungsprozess und steigert die allgemeine Benutzerzufriedenheit.

Zusammenfassend ist Genuine Captcha© nicht nur ein Werkzeug; es ist ein Paradigmenwechsel in unserem Ansatz zur Benutzerverifizierung - mit Priorität auf Datenschutz, Konformität und Benutzererfahrung in einer beispiellosen Weise.

Integratoren von CAPTCHA-Lösungen von Drittanbietern sollten sich der potenziellen rechtlichen Herausforderungen bewusst sein, denen sie bezüglich der Nichteinhaltung der DSGVO in Europa und ähnlicher Datenschutzvorschriften gegenüberstehen könnten. Die Sammlung und Verarbeitung personenbezogener Daten durch CAPTCHA-Systeme könnte, wenn sie nicht in Übereinstimmung mit diesen Gesetzen gehandhabt wird, zu erheblichen rechtlichen Konsequenzen führen, einschließlich hoher Geldstrafen und Reputationsschäden.

Genuine Captcha© bietet eine Lösung für das Dilemma, mit dem Integratoren von CAPTCHA-Lösungen von Drittanbietern konfrontiert sind, insbesondere in Bezug auf DSGVO-Konformität und Benutzerdatenschutz. Als Open-Source-Lösung ist es einzigartig darauf ausgelegt, menschliche Interaktionen zu verifizieren, ohne sich an der Sammlung privater Daten zu beteiligen.

Genuine Captcha© API: Integrationsleitfaden

Folgen Sie diesen einfachen Schritten, um unsere datenschutzorientierte CAPTCHA-Lösung zu integrieren

CAPTCHA generieren

Initiieren Sie eine CAPTCHA-Herausforderung für Ihre Benutzer.

Endpunkt: GET https://api.genuine-captcha.io/api/captcha/create


            fetch('https://api.genuine-captcha.io/api/captcha/create')
                .then(response => response.json())
                .then(data => {
                    const imageType = 'image/png'; // Oder 'image/jpeg', etc.
                    const base64Image = `data:${imageType};base64,${data.ImageAsBase64}`;
                    document.getElementById('captcha-image').src = base64Image;
            
                    try {
                        captchaSecret = data.SecretAsBase64;
                    } catch (error) {
                        console.error("Fehler beim Decodieren des Geheimnisses:", error);
                        // Fehler angemessen behandeln.
                    }
                });
                                

            {
                "ImageAsBase64": "[base64-kodiertes-Bild]",
                "SecretAsBase64": "[kodiertes-Geheimnis]"
            }

CAPTCHA lösen und verifizieren

Nachdem der Benutzer das CAPTCHA gelöst hat, verifizieren Sie seine Lösung.

Endpunkt: GET https://api.genuine-captcha.io/api/captcha/verify?captchaSolution=BENUTZER_LÖSUNG&captchaSecret=GEHEIMNIS_VON_CREATE


const captchaSolution = "21"; // Vom Benutzer bereitgestellte Lösung
const captchaSecret = "apiSecret"; // Vom Create-Aufruf erhaltenes Geheimnis, undekodiert

fetch(`https://api.genuine-captcha.io/api/captcha/verify?captchaSolution=${solution}&captchaSecret=${encodeURIComponent(captchaSecret)}`, {
    mode: 'cors' 
})      .then(response => {
        if (response.ok) {
            console.log("CAPTCHA erfolgreich verifiziert");
        } else {
            console.log("CAPTCHA-Verifizierung fehlgeschlagen");
        }
    });

Antworten:

  • HTTP 200: CAPTCHA erfolgreich verifiziert.
  • HTTP 400: CAPTCHA-Verifizierung fehlgeschlagen.
